Welcome to the API Tester

This interactive documentation allows you to test API endpoints directly from your browser. Select an endpoint from the sections below or navigate using the sidebar.

How to use:

  1. Navigate to the desired API section
  2. Fill in the required parameters
  3. Click "Send Request" to test the endpoint
  4. View the response in the panel above

Authentication

Authenticate with the API to access protected endpoints. The login endpoint will return a JWT token that will be automatically stored and used for subsequent requests.

POST /auth/login

Log in with email and password to get an authentication token

POST /auth/register

Register a new user account

Users
